# git-manager-appcast

Public companion to the private [Grim-Reaper/git-manager](https://github.com/Grim-Reaper/git-manager) repo. Exists solely to host the [Sparkle](https://sparkle-project.org) auto-update feed and the release zips it points at, both of which need to be reachable by unauthenticated clients.

- Stable channel: <https://grim-reaper.github.io/git-manager-appcast/appcast.xml>
- Nightly channel: <https://grim-reaper.github.io/git-manager-appcast/appcast-nightly.xml>

Release assets live under [Releases](https://github.com/Grim-Reaper/git-manager-appcast/releases). The publishing flow is `release.sh` / `nightly.sh` in the source repo.
